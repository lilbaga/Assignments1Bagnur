Event Management System
Программа для автоматизации учета мероприятий, управления составом участников и распределения ролей организаторов.
________________________________________________________________________
Описание проекта
Данная система реализована на языке Java и предназначена для работы с базой данных мероприятий. Основные функции включают в себя создание события с привязкой к календарной дате, расчет времени до начала мероприятия, а также фильтрацию и поиск участников.

Выполненные требования
1. Принципы ООП (Инкапсуляция, Наследование, Полиморфизм)

В проекте реализована иерархия классов на базе абстрактного класса Person.

Классы Organizer и Participant наследуют общие свойства (имя, фамилия) и расширяют их специфическими данными (департамент, email).

Полиморфизм реализован через переопределение абстрактного метода displayRole(), что позволяет вызывать специфичную логику для разных типов пользователей через общую ссылку.

2. Абстракция данных

Использование абстрактного класса Person позволяет выделить общие сущности предметной области, не допуская создания неопределенных объектов.

3. Работа с данными (Sorting, Searching)

Участники ивента организованы в коллекцию ArrayList.

Реализована сортировка списка участников по фамилии с использованием Comparator.

Добавлен метод поиска (findParticipant), выполняющий поиск по имени или фамилии без учета регистра.

Используется Java Time API для вычисления временных интервалов (количество дней до начала).

4. Переопределение методов Object

Для корректной работы с коллекциями и сравнения объектов были переопределены методы equals() и hashCode().

Метод toString() переопределен во всех классах для формирования структурированного текстового отчета.
